# v0.1.0 リリースノート草案

## 初回公開の概要

ChatGPTエクスポートの`conversations.json`をブラウザ内で解析し、会話の傾向をローカルで振り返る初回公開版です。入力ファイルは外部サービスへ送信せず、JSONファイルを手動選択して利用します。

## 主な機能

- 会話数、メッセージ数、ユーザー発言数、AI返信数の集計
- `mapping`と`current_node`を使った会話経路の復元
- 曜日×時間帯ヒートマップ、月別アクティビティ、頻出ワード
- Wrapped形式の総合サマリー
- 非同期チャンク処理、進捗表示、停止・再実行
- IndexedDBへの保存・読込・再保存・全削除
- 元の会話本文を含めないAI引き継ぎJSON v0.1
- ライト・ダークテーマ、レスポンシブ表示、キーボード操作とアクセシビリティ対応

## プライバシー設計

- 会話データはアプリ実行中に外部API、クラウドDB、分析サービスへ送信しません。
- 保存を選択した解析結果はブラウザ内のIndexedDBに保存します。共有PCでは利用後に保存データを削除してください。
- AI引き継ぎJSONには元の会話本文、メッセージ本文、会話タイトル、会話ID、元ファイル名、ローカルパスを出力しません。
- 頻出語などの派生集計情報は含まれるため、公開・共有前の内容確認が必要です。

## 既知の制限

- UIは日本語のみです。
- ZIP直接読込、期間比較、Web Workerは未対応です。
- 非常に大きな単一JSONでは`JSON.parse`中、非常に大きな単一会話では正規化中に停止操作が一時的に反映されない可能性があります。
- `next/font/google`により、ビルド時に外部フォント取得が発生する可能性があります。会話データはこの通信に含まれません。
- Next.js内包PostCSSの`GHSA-qx2v-qp2m-jg93` advisory（moderate）が存在します。現行アプリはユーザー入力をCSS処理へ渡さないため主な到達経路はビルド時です。stable版Next.jsで修正版が提供され次第、更新と再検証を行います。
